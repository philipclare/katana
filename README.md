# Example code for use on UNSW Katana Cluster Computer
## PBS Scripts and R code

This repository contains the PBS scripts and R code to show examples of how to use UNSW's HPC Katana.

Example 1 is a simply in-program parallel multiple imputation using the public data 'nhanes' and the function 'parlmice' from 'miceadds'.
Example 2 is a simple Monte-Carlo simulation, running a mixed effect model on simulated data 10,000 times.
